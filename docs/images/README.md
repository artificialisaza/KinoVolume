# Image Slots For GitHub

This folder is a stable place for screenshots, GIFs, and output examples used by the repository README.

Suggested structure:

- `interface/hero.gif` for a short animated hero preview
- `interface/main-window.png` for the main application screenshot
- `outputs/slice.png`
- `outputs/cuboid.png`
- `outputs/cylinder.png`
- `outputs/rings.png`
- `outputs/slittear.png`

Recommended practice:

- Export images at widths that still look good on GitHub without being excessively heavy.
- Prefer PNG for still screenshots and GIF only for short interface demonstrations.
- If you later want cleaner performance and smaller file size than GIF, switch to MP4 and link it separately from the README.
